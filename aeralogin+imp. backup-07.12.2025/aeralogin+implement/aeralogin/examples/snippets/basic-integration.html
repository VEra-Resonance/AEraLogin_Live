üíª Integration Code

Add the following snippet to your website to enable AEra Gate protection:

<!-- AEra Gate Integration -->
<script src="https://aeralogin.com/sdk/aera-gate.js"></script>
<script>
  AEraGate.init({
    clientId: 'your-client-id',
    autoProtect: true,
    onAuthenticated: (user) => {
      console.log('Welcome!', user.wallet);
    }
  });
</script>

Then mark any content with data-aera-protect to require authentication.
